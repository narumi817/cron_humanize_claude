# cron ビルダー設計（叩き台）

cron 式を UI 操作で生成する機能。

---

## 画面構成

```
┌─────────────────────────────────────┐
│  頻度を選ぶ（ラジオボタン）            │
│                                     │
│  ○ 毎分                             │
│  ○ N分ごと                          │
│  ○ 毎時                             │
│  ○ N時間ごと                         │
│  ○ 毎日                             │
│  ● 毎週  ← 選択中                    │
│  ○ 毎月                             │
│  ○ 毎年                             │
├─────────────────────────────────────┤
│  詳細設定（頻度に応じて切り替わる）     │
│                                     │
│  曜日:                              │
│  ☑ 月  □ 火  □ 水  □ 木  □ 金      │
│  □ 土  □ 日                        │
│                                     │
│  時刻:  [9] 時 [00] 分              │
├─────────────────────────────────────┤
│  生成された cron 式                  │
│  0 9 * * 1                         │
│                                     │
│  [ 変換する ]                        │
└─────────────────────────────────────┘
```

---

## 頻度ごとの詳細設定

| 頻度 | 詳細設定 UI | 生成例 |
|------|------------|--------|
| 毎分 | なし | `* * * * *` |
| N分ごと | 数値入力（1〜59） | `*/15 * * * *` |
| 毎時 | 分: 数値入力（0〜59） | `0 * * * *` |
| N時間ごと | 数値入力（1〜23）、分: 数値入力 | `0 */6 * * *` |
| 毎日 | 時・分: 数値入力 or セレクト | `0 9 * * *` |
| 毎週 | 曜日: チェックボックス（複数可）、時・分 | `0 9 * * 1,5` |
| 毎月 | 日: テキスト入力（例: `1,15`）、時・分 | `0 9 1,15 * *` |
| 毎年 | 月: テキスト入力（例: `4,10`）、日、時・分 | `0 9 1 4,10 *` |

---

## 各項目の UI コンポーネント案

### 曜日（毎週）
チェックボックスで複数選択。

```
□ 月（1）  □ 火（2）  □ 水（3）  □ 木（4）  □ 金（5）  □ 土（6）  □ 日（0）
```

### 日（毎月）
テキスト入力でカンマ区切り入力 or セレクトボックス複数選択。

```
日: [ 1,15 ]  ← 例: 1日と15日
```

### 月（毎年）
テキスト入力でカンマ区切り入力 or チェックボックス。

```
月: [ 4,10 ]  ← 例: 4月と10月
```

### 時・分
数値入力（セレクトボックスも可）。

```
時: [9]  分: [00]
```

---

## 実装方針

- 頻度のラジオボタン切り替えは Turbo（Rails 標準）で詳細設定を差し替える
  - JS を最小限にしたい場合は form submit + 部分テンプレートで実現可能
- 入力値から cron 式への変換は Service Object（`CronBuilderService`）に切り出す
- バリデーションは既存の `CronHumanizeService` で流用（生成後に検証）
