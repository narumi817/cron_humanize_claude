# cron ビルダー設計

cron 式を UI 操作で生成する機能。

---

## 画面構成

```
┌─────────────────────────────────────┐
│  頻度を選ぶ（ラジオボタン）            │
│                                     │
│  ○ 毎分                             │
│  ○ N分ごと                          │
│  ○ 毎時                             │
│  ○ N時間ごと                         │
│  ○ 毎日                             │
│  ● 毎週  ← 選択中                    │
│  ○ 毎月                             │
│  ○ 毎年                             │
├─────────────────────────────────────┤
│  詳細設定（頻度に応じて切り替わる）     │
│                                     │
│  曜日: ☑月 □火 □水 □木 ☑金 □土 □日 │
│  時刻:  [ 9 ] 時 [ 00 ] 分          │
├─────────────────────────────────────┤
│  生成された cron 式                  │
│  0 9 * * 1,5                       │
├─────────────────────────────────────┤
│  今後5回のスケジュール                │
│  2026年6月9日(火) 09:00             │
│  2026年6月12日(金) 09:00            │
│  ...                                │
└─────────────────────────────────────┘
```

---

## 頻度ごとの詳細設定

曜日はチェックボックス、それ以外の詳細入力はテキスト入力。複数指定はカンマ区切り。

| 頻度 | 詳細設定 | 入力例 | 生成される cron 式 |
|------|---------|--------|-------------------|
| 毎分 | なし | — | `* * * * *` |
| N分ごと | 分の間隔 | `15` | `*/15 * * * *` |
| 毎時 | 分 | `0` | `0 * * * *` |
| N時間ごと | 時間の間隔、分 | `6` / `0` | `0 */6 * * *` |
| 毎日 | 時・分 | `9` / `0` | `0 9 * * *` |
| 毎週 | 曜日: チェックボックス（複数可）、時・分 | 月・金 / `9` / `0` | `0 9 * * 1,5` |
| 毎月 | 日（1〜31）、時・分 | `1,15` / `9` / `0` | `0 9 1,15 * *` |
| 毎年 | 月（1〜12）、日、時・分 | `4,10` / `1` / `9` / `0` | `0 9 1 4,10 *` |

---

## 入力ミスへの対処

テキスト入力は自由度が高い反面、入力ミスが起きやすい。
cron 式の下に**今後5回のスケジュール**を表示することで、ユーザーが意図通りかを自分で確認できる。

- 既存の `CronHumanizeService` をそのまま流用できる
- 「2026年6月9日(土)」が出てきたら「あ、曜日がずれてる」とすぐ気づける

---

## 実装方針

- 頻度のラジオボタン切り替えは Turbo（Rails 標準）で詳細設定部分を差し替える
- 入力値から cron 式への変換は `CronBuilderService` に切り出す
- 生成した cron 式を既存の `CronHumanizeService` に渡して今後5回を表示する
- バリデーションは `CronHumanizeService`（Fugit）で流用

---

## 決定事項

- 頻度の選択: ラジオボタン
- 詳細入力: 曜日はチェックボックス、それ以外はテキスト入力（カンマ区切りで複数指定）
- 入力ミス対策: 今後5回のスケジュール表示で自己確認
